# StaticValidationGeometries

This folder contains the code and supporting files for the static validation geometry cases used in the manuscript archive `ddg_curved_volume_manuscript_cases_v45`.

## Manuscript relevance

This folder corresponds to the static validation cases reported in the manuscript.

The main file used for the manuscript results in this folder is:

`manuscript_curved_volume_final.ipynb`

## Main files

### Main notebook
- `manuscript_curved_volume_final.ipynb`  
  Main notebook used to generate the manuscript results for the static validation geometry cases.

### Direct local dependency
- `_curved_volume.py`  
  Helper module used by the main notebook.

### Additional folders
- `mesh/`  
  Mesh files used for the validation geometries.

- `curved_volume/`  
  Supporting curved-volume related files.

- `neatplot-main/`  
  Plotting / figure support files used in the workflow.

## Recommended execution

Open and run the notebook from within this folder so that the local module and supporting files can be found correctly.

## Notes

- This archive is intended to preserve the exact code snapshot associated with manuscript version `v45`.
- File names were kept close to the original development naming for internal traceability.
- For publication citation, please cite the archived Zenodo record corresponding to this code snapshot.
